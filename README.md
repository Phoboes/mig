# Whale Watching App

## Setup Steps

- Clone this repo
- `yarn install`
- `yarn prepare`
- Adding environment variables:
  - Create a new file called `.env.local`
  - Copy the content from `.env.local.example` and paste it into `.env.local`
  - Change the values
